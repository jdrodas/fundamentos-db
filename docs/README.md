# Fundamentos de Bases de Datos Relacionales

Bienvenido al repositorio de recursos del curso. Aquí encontrarás los scripts SQL
de referencia, los datasets de prueba y las guías para cada unidad.

## ¿Cómo usar estos recursos?

1. Navega por las unidades usando el menú lateral.
2. Cada unidad tiene sus objetivos, criterios de evaluación y recursos listos para descargar.
3. Los scripts están numerados en orden de ejecución: comienza siempre por `00_schema.sql`.

## Motor de base de datos

Todos los scripts están escritos para **PostgreSQL 16+**.
